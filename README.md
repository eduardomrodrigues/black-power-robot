# black-power-robot

> A GitHub App built with [Probot](https://github.com/probot/probot) that npx create-probot-app my-first-app

## Setup

```sh
# Install dependencies
npm install

# Run with hot reload
npm run build:watch

# Compile and run
npm run build
npm run start
```
